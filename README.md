## O MINI SISTEMA DE COBRANÇAS POR EMAIL

* EU USEI O N8N - (para automação)
* GOOGLE SHEET - (como banco de dados)
* GITHUB PAGES - (como hospedagem dá página)

O sistema funciona assim, eu tenho a planilha de pessoas de contas a receber, com as informações dos clientes como nome email e etc.
eu salvo essa planilha em xlsx, xls ou csv, e arrastou para o campo de enviar dá página, coloco o email principal e seleciono a opção de
inadinplentes, ai a automação começa a disparar os emails se tem uma lista de 1000 emails de devedores ele vai enviar para todos os 
emails em segundos.
